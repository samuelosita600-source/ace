# ACE

ACE is an AI Operating System.

It is NOT a chatbot.

Claude is NOT ACE.

Claude is only the reasoning engine.

ACE owns:

- Memory
- Relationships
- User Profile
- Goals
- Context
- Skills
- Reflection
- Advice
- Traits

The system should always retrieve context dynamically.

Never place everything into one giant prompt.

Every engine must have one responsibility.

Every feature should be modular.

Every module should be independently testable.

UI Philosophy:

Premium.

Minimal.

Modern.

Apple-quality.

Smooth animations.

Large spacing.

Rounded corners.

Black and white design.

The avatar will become the center of the experience.

Architecture Philosophy:

Context Builder

↓

Prompt Orchestrator

↓

Claude

↓

Reflection

↓

Memory Update

All code should follow this architecture.
