# ACE Core Engine

Version: 1.0.0

Status: Designing

---

## Purpose

The Core Engine Framework provides the shared functionality that every engine in ACE inherits.

It ensures that all engines behave consistently and follow the same lifecycle.

---

## Responsibilities

- Provide a common engine lifecycle.
- Store engine metadata.
- Track engine status.
- Serve as the parent class for all engines.

---

## Current Features

- Engine name
- Engine version
- Start method
- Stop method
- Status tracking
