# ACE Entity Extraction Engine

Version: 1.0.0

Status: Designing

---

## Purpose

The Entity Engine extracts important entities from memories and conversations.

Entities help ACE understand people, places, dates, organizations, events, and relationships.

---

## Current Features

- Person detection
- Relationship detection
- Event detection

---

## Future Features

- AI extraction
- Name detection
- Date extraction
- Time extraction
- Organization detection
- Location detection
- Sentiment-aware entities
- Custom entity types
