# ACE Memory Pipeline

Version: 1.0.0

Status: Designing

---

## Purpose

The Memory Pipeline evaluates information before it becomes a memory.

It prevents unnecessary information from filling ACE's memory.

---

## Current Steps

1. Receive candidate memory
2. Decide whether to remember it
3. Calculate importance
4. Pass approved memories to the Memory Engine

---

## Future Features

- Emotion detection
