# ACE Event System

Version: 1.0.0

Status: Designing

---

## Purpose

The Event Bus allows engines to communicate without directly depending on one another.

---

## Benefits

- Loose coupling
- Easier testing
- Better scalability
- Cleaner architecture

---

## Example Events

- MessageReceived
- MessageSent
- MemoryCreated
- MemoryUpdated
- ProfileUpdated
- RelationshipUpdated
- AdviceGenerated
- GoalUpdated
