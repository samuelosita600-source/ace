# ACE Prompt Orchestrator

Version: 1.0.0

Status: Designing

---

## Purpose

The Prompt Orchestrator gathers all relevant context before an AI model is called.

Instead of using one massive prompt, it dynamically builds a prompt using information from ACE's engines.

## Current Sources

- Identity Engine
- User Engine
- Memory Engine
